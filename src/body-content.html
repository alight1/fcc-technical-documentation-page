Introduction

What should this document? After all, there are plenty of low-cost documentation services online (Atlassian makes it easy) and there are surely better ways to spend time coding.

This is a project to demonstrate my knowledge of responsive HTML and CSS ;-). I plan to document how I use AWS S3 to host my static websites at a very low cost (much lower than Webflow site plans) while having more control than most free hosting services offer.

Of course, Webflow and similar paid services offer far more value than simple website hosting, so the costs are justified for many companies. For simple code portfolio projects, AWS S3 + CloudFront + Certificate Manager provide an excellent alternative for those who know how to code, while Webpack and open-source plugins automate publishing.

Note: most teams of any size would benefit from a robust CI/CD workflow. This is not that.
